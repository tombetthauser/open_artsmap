Welcome to the Open Arts Map project landing page!

An open map resource built to help artists, galleries and arts institutions find eachother. Not for profit, made for artists by artists.

If you'd like to have your studio, gallery, collective, arts institution etc. added to the map we have a Google Form set up for the time being. If / when the project gains momentum a public database and api will be set up to let artists manage their map entry directly and let arts groups use


full-screen map

This project was developed in Sacramento, CA with support from a group of graduates of the San Francisco Art Institute and the Yale School of Art. The code associated with this project is freely available to run, copy, distribute, study, change and improve via github (link below). GitHub Repository
